Fruit Shop 水果捞商店
===

####提示：
本项目基于intellij idea的专业版<br>
使用ssm框架技术<br>
layui模块化前端<br>

## 一、运行试用
* 首先，把项目clone到本地。
* 接着，用idea把项目打开。
